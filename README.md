# Chatto Metro Half Marathon 2026

Production website for the Chatto Metro Half Marathon 2026.

## Development
npm install
npm run dev

## Build
npm run build
npm run preview

## Deploy (Vercel)
Build command: astro build
Output directory: dist
Install command: npm install

## Vercel DNS (Hostinger -> triathlonbangladesh.com)
Add these records in Hostinger DNS:
- A record: @ -> 76.76.21.21
- CNAME: www -> cname.vercel-dns.com
